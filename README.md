# YouTube Downloader (Clean & Modern)

A modern desktop YouTube downloader built with **Python + PySide6 + yt-dlp**.

Download:
- **Single videos**
- **Playlists**
- **Channels**
- **Audio-only** (MP3 / M4A / Opus)

Includes **Light/Dark mode**, **Windows-friendly quality presets**, **skip existing files**, **multithreaded downloads**, **FFmpeg auto-detection + manual path**, **notifications**, and a **copyable debug log** for bug reports.

> ⚠️ This project is for personal use. Respect creators and your local laws/terms of service.

---

## Features

✅ Modern UI (PySide6)  
✅ Light & Dark mode (toggle + saved)  
✅ Video: Single / Playlist / Channel  
✅ Audio-only: MP3 / M4A / Opus  
✅ Quality presets (including Windows-friendly MP4/H.264 preference)  
✅ Skip existing files (no re-downloads)  
✅ Multithreaded playlist/channel downloading  
✅ FFmpeg detection + manual FFmpeg path setting  
✅ System notifications on completion/failure  
✅ Copyable debug log for GitHub issues  

---

## Requirements

- **Python 3.10+** recommended
- **FFmpeg** (required for merging streams + audio extraction)
- Windows / macOS / Linux supported (Windows easiest with FFmpeg path setting)
